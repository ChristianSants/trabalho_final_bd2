# Projeto Final da cadeira de Banco de dados II

Um simulador de recuperação de falhas de um banco de dados.


## Funcionalidades

- Iniciar transação
- Updates nas transações
- Finalizar transação
- Inserir commit
- Inserir checkpoint
- Simular falha
- Visualizar log disco/memória
- Visualizar banco disco/memória


## Stack utilizada

**Front-end:** Bootstrap, JavaScript e JQuery

**Back-end:** PHP
